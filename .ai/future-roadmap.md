# Future Roadmap

## Phase 1 — Brand Website Foundation
- Build core route set (`/`, `/about`, `/services`, `/therapist`, `/faq`, `/contact`).
- Implement design token system and reusable component architecture.
- Establish trust-first copy and metadata baselines.

## Phase 2 — Booking Funnel
- Add `/book` route and low-friction booking flow.
- Introduce scheduling UX with clear expectations and consent language.
- Optimize conversion while preserving calm experience.

## Phase 3 — SEO Content Engine
- Launch `/blog` with structured editorial templates.
- Publish high-intent bilingual-support articles.
- Build topic clusters around Turkish-language therapy in UK.

## Phase 4 — CMS Integration
- Introduce headless CMS for content-managed pages.
- Define robust schemas for sections, FAQs, and articles.
- Add preview and editorial QA workflow.

## Phase 5 — Growth Systems
- Implement ethical analytics and funnel instrumentation.
- Run trust-preserving UX experiments.
- Improve conversion pathways without pressure tactics.

## Phase 6 — Automation + CRM
- Integrate enquiries with CRM pipeline.
- Automate response workflows while preserving privacy safeguards.
- Add operational dashboards for lead quality and response SLAs.

## Ongoing Guardrails
Across all phases:
- anxiety-reducing UX is mandatory
- accessibility and privacy remain first-class requirements
- brand consistency enforced through tokens and component reuse
